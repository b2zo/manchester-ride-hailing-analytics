# Manchester Ride-Hailing Analytics

A data analytics project that analyses ride-hailing profitability across Greater Manchester.

## Objectives

- Identify the most profitable pickup zones
- Compare airport and non-airport trips
- Analyse earnings per hour
- Build dashboards and visualisations
- Demonstrate Python, SQL and Power BI skills

## Technologies

- Python
- Pandas
- SQL
- Power BI
- Git
- GitHub
